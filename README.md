# 災害廃棄物メッシュ閲覧ツール (Disaster Waste Mesh Viewer)

災害廃棄物量が記録されたCSVファイルを読み込み、地図上にメッシュとして可視化するツールです。
ブラウザ上で動作し、サーバーサイドの処理は必要ありません。

## 機能
- **CSV読み込み**: ローカルのCSVファイル (`CODE`, `Value`列を含む) を読み込みます。
- **地図表示**: Leaflet.js と国土地理院タイルを使用。
- **データ可視化**: 10桁メッシュコードをポリゴンに変換し、廃棄物量に応じて色分け表示します。
- **インタラクティブ**: メッシュをクリックまたはホバーすると詳細情報を表示します。

## 使い方
1. 画面左上の「ファイルを選択」ボタンをクリックします。
2. 対象のCSVファイル（例: `DebrisET_23_R02R05.csv`）を選択します。
3. 読み込みが完了すると、地図上にデータが表示されます。

### 対応CSVフォーマット
ヘッダー行が必要です。
```csv
CODE,Value
5236556342,9.84
...
```
- `CODE`: 10桁のメッシュコード (半角数字)
- `Value`: 廃棄物量 (数値)

## 動作環境
- 最新のモダンブラウザ (Chrome, Edge, Firefox, Safari)

## ローカルでの実行
```bash
# 依存関係はありません。index.html をブラウザで開くだけでも動作しますが、
# ローカルサーバー経由での実行を推奨します。
npx http-server .
```

## GitHub Pages へのデプロイ方法
1. このリポジトリをGitHubにPushします。
2. GitHubリポジトリの **Settings** > **Pages** を開きます。
3. **Source** セクションで `Deploy from a branch` を選択します。
4. **Branch** で `master` (または `main`) と `/ (root)` を選択し、**Save** をクリックします。
5. しばらくすると、表示されたURLでツールが公開されます。
