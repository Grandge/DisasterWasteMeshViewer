# 災害廃棄物メッシュ閲覧ツール (Disaster Waste Mesh Viewer)

災害廃棄物量が記録されたCSVファイルを読み込み、地図上にメッシュとして可視化するツールです。
ブラウザ上で動作し、サーバーサイドの処理は必要ありません。

## 公開URL
https://Grandge.github.io/DisasterWasteMeshViewer/

## 機能
- **CSV読み込み**: ローカルのCSVファイル (`CODE`, `Value`列を含む) を読み込みます。
- **大容量データ対応**: 1万件を超えるデータの場合、第2次メッシュ単位で表示エリアを選択可能です。
- **地図表示**: Leaflet.js と国土地理院タイルを使用。
- **データ可視化**: 10桁メッシュコードをポリゴンに変換し、廃棄物量に応じて色分け表示します。

## 使い方
1. 上記URLにアクセスするか、ローカルで `index.html` を開きます。
2. 画面左上の「ファイルを選択」ボタンをクリックします。
3. 対象のCSVファイル（例: `DebrisET_23_R02R05.csv`）を選択します。
4. データ量が多い場合はエリア選択画面が表示されるので、表示したいエリアを選択してください。
